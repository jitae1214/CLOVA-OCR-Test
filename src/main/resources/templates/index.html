이미지에서 텍스트를 추출하는 스마트한 방법

설정 방법

클로바 OCR을 사용하려면 src/main/resources/application.properties 파일에 다음 설정을 추가해주세요:

clova.ocr.api-url=https://your-api-gateway-url
clova.ocr.secret-key=your-secret-key
네이버 클라우드 플랫폼에서 클로바 OCR 서비스를 신청하고 API Gateway URL과 Secret Key를 발급받아 설정하세요.

이미지를 업로드하세요

클릭하여 파일을 선택하거나 드래그 앤 드롭으로 업로드하세요

지원 파일 형식: JPG, PNG, GIF, BMP 등의 이미지 파일
